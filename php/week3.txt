$_POST와 $_SERVER는 모두 PHP에서 제공하는 **미리 정의된 전역 배열(Superglobal Array)**이지만, 그 목적과 용도가 다릅니다.

1. $_POST
	역할: POST 방식으로 전송된 폼 데이터를 저장하는 배열입니다.
	용도: 클라이언트가 HTML 폼 등을 통해 서버로 데이터를 전송할 때, 그 데이터를 서버에서 받아오는 데 사용됩니다.
	예시:
		
		$name = $_POST['name'];  // 사용자가 폼에 입력한 'name' 데이터를 받음
	주로 사용되는 상황:
		로그인 폼
		회원 가입 폼
		데이터 입력/업로드와 같은 작업에서 데이터를 서버로 전달할 때
2. $_SERVER
	역할: 서버와 실행 환경에 대한 정보를 제공하는 배열입니다.
	용도: 요청과 관련된 정보(예: HTTP 요청 방식, 요청 URL, 사용자 에이전트 등)를 제공하며, 서버의 설정 및 상태를 확인할 수 있습니다.
	예시:
	
		$method = $_SERVER['REQUEST_METHOD'];  // GET, POST, PUT 등 요청 방식을 받음
		$userAgent = $_SERVER['HTTP_USER_AGENT'];  // 클라이언트의 브라우저 정보
	주로 사용되는 상황:
		요청 방식(GET, POST) 확인
		클라이언트의 IP 주소 또는 브라우저 정보 확인
		서버 정보나 스크립트 경로에 대한 정보 확인

- 주요 차이점
**$_POST**는 클라이언트가 POST 요청을 통해 보낸 데이터를 다루고,
**$_SERVER**는 서버나 환경 설정과 관련된 정보를 제공합니다.
둘 다 요청 처리에 중요하지만, 그 역할이 완전히 다릅니다. $_POST는 클라이언트 데이터를 처리하는 데 사용되고, $_SERVER는 요청과 서버 정보를 확인하는 데 사용됩니다.
